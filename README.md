# AthleteMatch

College soccer matching for players comparing NCAA programs by academic range, soccer level, division, region, and gender-specific program availability.

## What It Does

- Scores schools against GPA, SAT or ACT, club level, playing time, division, and region preferences.
- Separates men's and women's soccer sponsorship so schools only appear when that program exists.
- Uses a generated NCAA Directory data set covering D1, D2, and D3 soccer programs.
- Shows fit reasons, score breakdowns, school links, and program links.

## Data

`data/schools.json` is generated by `scripts/generate-school-data.mjs` from NCAA Directory sport sponsorship lists.

Current generated totals:

- 1,037 NCAA schools
- 823 men's soccer programs
- 1,028 women's soccer programs
- D1, D2, and D3 included

Run this to refresh the database:

```bash
node scripts/generate-school-data.mjs
```
